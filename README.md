===========================================
   SC2 Fog of War Remover v2.0
   Full Map Reveal for StarCraft II
===========================================

УСТАНОВКА:
1. Установить .NET 8.0 SDK: https://dotnet.microsoft.com/download
2. Запустить build.bat (компиляция)
3. Запустить StarCraft II
4. Запустить run.bat (от имени администратора)

РАБОТАЕТ В:
- Оффлайн режиме / против AI
- Кастомных играх (игротеках)
- Онлайн матчах (НО есть риск бана!)

НЕ РАБОТАЕТ:
- Если игра не запущена
- Если запущена без прав администратора
- Если версия игры сильно устарела

ЧТО ДЕЛАЕТ:
- Убирает туман войны полностью
- Вся карта всегда видна
- Работает через патчинг памяти

ПРИМЕЧАНИЕ:
- При закрытии программы туман возвращается
- Для онлайн-игры использовать на свой страх и риск

ОШИБКИ И РЕШЕНИЯ:

Ошибка "NETSDK1136":
→ Установите .NET 8.0 SDK с официального сайта

Ошибка "Access denied":
→ Запустите run.bat от имени администратора (ПКМ -> Run as admin)

Ошибка "SC2 not found":
→ Убедитесь, что StarCraft II запущен до запуска программы

Туман не исчезает:
→ Перезапустите игру и программу
→ Возможно, вышло обновление игры (нужны новые сигнатуры)
