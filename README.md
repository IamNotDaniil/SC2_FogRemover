# SC2 Fog of War Remover

Простой и эффективный инструмент для снятия тумана войны в StarCraft II: Legacy of the Void.

## Как это работает

Инструмент патчит память процесса SC2_x64.exe, заменяя условные переходы в функциях рендера тумана войны на безусловные, а также переопределяет BlendState в DirectX 11.

## Установка

1. Установи .NET 8.0 SDK: https://dotnet.microsoft.com/download
2. Скачай или склонируй репозиторий
3. Запусти `build.bat`

## Использование

1. Запусти StarCraft II
2. Запусти `run.bat` **от имени администратора**
3. Туман войны исчезнет

## Требования

- Windows 10/11 x64
- StarCraft II: Legacy of the Void (последний патч)
- .NET 8.0 Runtime
- Запуск от администратора

## Важно

Использование в онлайн-матчах нарушает правила Blizzard Entertainment.
Только для одиночной игры и образовательных целей.
